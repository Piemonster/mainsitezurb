TableSurf

TableSurf.in connects people who want to cook with people who want to eat. We aim to reduce food wastage and encourage community spirit by getting people together to share meals. We have tried to make it as easy as possible to import your friends from multiple sources to get up and running quickly. Why not see if anyone is hungry in your area?

Technologies

  • Field Autocompletes
  • OAuth
  • REST API
  • Contacts Import
  • Geolocation
  • Yummly integration

Status

Online

TableSurf Single Page

We are building a frontend for TableSurf.in that consumes the API provided by the current system. It is hoped this will enable us to leverage tools such as PhoneGap to provide a native experience on phones but maintaining one unified codebase. The quality of the application will be greatly increased when we have finished. We hope to by live by the end of February.
